Módulo para pór etiquetas simples para ubicar productos. # Descripción Las organizaciones más pequeñas pueden requerir alguna forma de organizar la colocación de los productos en las ubicaciones de su almacen, pero no poseen estructura burocrática suficiente para usar adecuadamente las ubicaciones de Odoo, más allá de ubicaciones específicas tales como Devoluciones o Deshechos. # Uso Después de instalar el módulo en la ficha de productos aparecerá un campo para la etiqueta de producto. Dicho campo es visible también en la vista de lista de productos, y también es posible agrupar por ubicación.
